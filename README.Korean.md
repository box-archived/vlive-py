vlivepy는 파이썬 기반의 비공식 [VLIVE](https://www.vlive.tv/) API입니다.

## 설치
[PyPI](https://pypi.org/project/vlivepy/) 를 통해 설치할 수 있습니다.
```console
$ python -m pip install vlivepy
```
